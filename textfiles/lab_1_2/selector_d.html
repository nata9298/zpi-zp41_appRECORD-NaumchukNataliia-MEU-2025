Селектори в CSS

Селектори в CSS дозволяють вибирати HTML-елементи для застосування стилів. Вони визначають, до яких елементів на сторінці будуть застосовані конкретні стилі. Ось основні типи селекторів:

1. Селектор за тегом (Type Selector)

Цей селектор застосовується до всіх елементів конкретного типу. Наприклад, селектор p вибирає всі параграфи на сторінці.

p { color: blue; }

2. Селектор за класом (Class Selector)

Селектор за класом дозволяє вибирати елементи, що мають певний клас. Клас вказується після крапки. Цей селектор використовується для групи елементів, що мають спільний клас.

.example { font-size: 16px; }

3. Селектор за ідентифікатором (ID Selector)

Селектор за ідентифікатором вибирає єдиний елемент на сторінці з певним id. Ідентифікатор вказується після символу #.

#header { background-color: gray; }

4. Груповий селектор (Group Selector)

Груповий селектор дозволяє застосовувати один набір стилів до кількох елементів одночасно. Елементи розділяються комою.

h1, h2, p { margin: 0; }

5. Селектор на основі атрибутів (Attribute Selector)

Цей селектор вибирає елементи за значенням їх атрибутів. Наприклад, селектор вибирає всі посилання з атрибутом href.

a[href] { color: green; }

Висновок

CSS-селектори — це потужний інструмент для стилізації елементів на веб-сторінках. Вибір правильного селектора залежить від того, чи потрібно ви змінити один елемент, групу елементів чи всі елементи одного типу.
