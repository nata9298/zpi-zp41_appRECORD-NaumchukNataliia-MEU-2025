Просунутий CSS:

У стилях використовуються техніки просунутого CSS для поліпшення візуального вигляду та функціональності:

1. Градієнти:

Активно застосовуються лінійні градієнти через background: linear-gradient(...) для кнопок, секцій і фону (.cta-button, .test-section, .login-button).

2. Анімації та переходи:

Застосовані плавні переходи через transition, наприклад, для кнопок (.cta-button:hover, nav a:hover) при зміні кольору, розміру або додаванні тіней.

3. Тіні:

Використовуються box-shadow для створення глибини та об'єму елементів, наприклад, для секцій (.section, .footer, .teacher).

4. Медійні запити:

Застосовано медіа-запити (@media) для адаптивності, наприклад, в секції @media (max-width: 500px), яка змінює розміри зображення в секції .hero.

5. Ефекти наведення:

Кнопки та посилання мають інтерактивні ефекти через стилі :hover, наприклад, збільшення розміру (transform: scale(1.05)) та додавання тіні.

6. Адаптивність:

Гнучкі макети реалізовано через flexbox для елементів (.hero, .teacher-list, header), що дозволяє елементам коректно розташовуватись на різних екранах.

7. Складні рамки:

Використовуються властивості border-radius для закруглення кутів (.hero, .section, .teacher) та градієнтні рамки через додаткові стилі (наприклад, .section-content).
