Способи підключення стилів в HTML

В HTML існує кілька способів підключення стилів, які використовуються для оформлення веб-сторінок:

1. Вбудовані стилі (inline styles)

Вбудовані стилі додаються безпосередньо в атрибут style конкретного елемента HTML. Це дозволяє швидко оформити окремий елемент.

Приклад:

<p style="color: red;">Це текст червоного кольору.</p>

Переваги: простота використання для невеликих змін. Недоліки: ускладнюють читання та масштабування коду.

2. Внутрішні стилі (internal styles)

Внутрішні стилі додаються в спеціальний тег <style> всередині секції <head> HTML-документа. Вони застосовуються до всієї сторінки.

Приклад:

<style>
body {
    background-color: lightblue;
}
h1 {
    color: green;
}
</style>

Переваги: дозволяють легко керувати стилями на одній сторінці. Недоліки: не підходять для багаторазового використання на різних сторінках.

3. Зовнішні стилі (external styles)

Зовнішні стилі зберігаються у окремому CSS-файлі, який підключається до HTML-документа за допомогою тега <link>. Це найкращий підхід для великих проєктів.

Приклад:

<link rel="stylesheet" href="styles.css">

Переваги: дозволяють використовувати одні й ті ж стилі на кількох сторінках. Недоліки: потрібне додаткове завантаження файлів.

Висновок

Вибір способу підключення стилів залежить від конкретної задачі. Для простих змін підходять вбудовані стилі, для сторінок з унікальним оформленням — внутрішні стилі, а для великих проєктів найкращим вибором є зовнішні CSS-файли.
